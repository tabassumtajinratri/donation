Important FAQ

What is the Document Object Model(DOM)?

The HTML DOM is a standard object model and programming interface for HTML

How Do you select an element from the DOM?

Selecting by ID: let element = document.getElementById("elementId");

Selecting by Class: let element = document.getElementsByClassName("className");;

Selecting by Tag Name: let element = document.querySelector(".className");;

Query Selector All (Multiple Elements): let elements = document.querySelectorAll(".className");

Query Selector (Single Element): let element = document.querySelector(".className");

What is the event delegation in the context of the DOM, and why is it usefull?

Event delegation is a JavaScript technique where add a single event listener to a parent element to manage events triggered by its child elements.

Event delegation is useful because it improves performance and simplifies code by using a single event listener to manage multiple child elements, including dynamically added ones.

How do you manipulate an element's attributes and styles using DOM?
